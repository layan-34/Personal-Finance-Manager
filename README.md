# Personal Finance App

A simple and efficient Android application to manage your daily income, expenses, and budgets. Built with Java and SQLite.

## Features

*   **Dashboard**: View your total income, expenses, and current balance.
*   **Transactions**: Add, edit, and delete daily financial records.
*   **Budgets**: Set monthly spending limits for different categories.
*   **Goals**: Track your savings progress for future goals.
*   **Reports**: View summaries by day, week, or month.
*   **Charts**: Visual breakdown of your data using Pie and Bar charts.

## Tech Stack

*   **Language**: Java
*   **Architecture**: MVVM (Model-View-ViewModel)
*   **Database**: Raw SQLite (No external libraries)
*   **UI**: XML & Material Design

## How to Run

1.  Clone this repository.
2.  Open the project in **Android Studio**.
3.  Run on an Emulator or Device.
